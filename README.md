# Medicus

## Backend instructions:

### Run `pip install flask`

### On Windows:

- On the command line:
- [x] `set FLASK_APP=flask_server.py`
- [x] `set FLASK_ENV=development`

### On Linux / Mac:

- On the command line:
- [x] `export FLASK_APP=flash_server.py`
- [x] `export FLASK_ENV=development`

### Run `flask run`
